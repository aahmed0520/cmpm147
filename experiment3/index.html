Experiment X - Procedural Worlds

seed

Description

This project explores procedural generation through the creation of a dungeon and overworld system using tiles.

Technical

Includes grid-based terrain generation, autotiling, seeded randomness, animation, and layered rendering techniques.

Reflection

Using p5.js, I learned to generate worlds using noise and random values and how to lay out tiles dynamically. Combining dungeon and overworld generators helped me understand system design and modularity in creative coding.
