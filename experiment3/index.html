Experiment X - Procedural Worlds

Description

This project explores procedural generation through the creation of a dungeon and overworld system using tiles.

Technical

Includes grid-based terrain generation, autotiling, seeded randomness, animation, and layered environmental effects like clouds.

Reflection

This project challenged me to learn noise-based generation and dynamic rendering using p5.js. Tweaking tile logic and coordinating multiple modes (dungeon vs. overworld) taught me how to organize code better and push beyond a static sketch.
