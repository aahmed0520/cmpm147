Experiment 4 - Infinite Desert

Description

This generator creates an infinite desert terrain with rocky, dusty, and dune-based regions that shift based on spatial and key-based variation. The tiles respond to clicks and animate with drifting sand over time.

Technical

The world uses isometric tiles with height variation via noise. The terrain texture is procedurally generated, and time-based animation adds dynamic ripples to dune tiles. Each world key produces a unique yet reproducible terrain layout using xxHash.

Reflection

This generator was built off the Infinite Worlds project base. I extended it with height illusion, animated dunes, click-reactive pulses, and multiple tile textures. Moving it to the portfolio required blending engine.js and my_world.js into one sketch, and ensuring dynamic canvas resizing worked properly with jQuery and p5.js integration.
