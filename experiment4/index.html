Experiment 4 - Infinite Worlds

Description

This project features three interactive infinite generators: a desert, a critter crawler, and a building combiner. Each can be toggled using the buttons above.

Technical

Each sketch is a standalone p5 instance using seeded noise, procedural generation, and asset-free tile rendering. Clicks interact with the environment. World keys are hashed using xxHash for reproducibility.

Reflection

I separated the generators into individual engine-powered sketches and loaded them via buttons using p5 instance mode. Getting all three to work cleanly in a shared layout required isolating state, properly resetting DOM elements, and keeping the key input consistent.
