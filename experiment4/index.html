Experiment 4 - Infinite Desert

Description

Arrow keys scroll. Clicking changes tiles.
This generator creates an infinite desert terrain using isometric tiles and procedural terrain types.

Technical

The world uses noise to determine tile types (sand, rock, dust, dune) and applies animated ripple effects based on time. Clicks are tracked per tile using a timestamp. Each world is seeded using a key hashed via xxHash for repeatability.

Reflection

I merged engine.js and my_world.js into one `sketch_desert.js` for my portfolio. Moving this over required layout fixes and separating the input and canvas from the description box. It now cleanly integrates into the portfolio format.
