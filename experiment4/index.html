Experiment 4 - Infinite Worlds

Description

This generator includes both a desert terrain and a critter crawler that follows apples. Use the buttons above to switch between sketches.

Technical

Each sketch is powered by p5 instance mode and uses procedural noise-based generation. Swapping sketches requires careful DOM and canvas cleanup.

Reflection

I structured this project to allow smooth toggling between independent sketches using a shared canvas layout. This required proper use of `new p5(...)` instance setup and cleanup handling.
