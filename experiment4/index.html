Experiment 4 - Infinite Worlds

Description

This page showcases three infinite world generators developed in p5.js. Toggle between them to explore each unique variation.

Technical

Each generator is implemented in a separate file and run in isolation using p5.js instances. The worlds respond to key input and mouse clicks, and allow for user-controlled seeds to deterministically generate infinite tilemaps.

Reflection

This experiment allowed me to explore variation and interactivity in infinite procedural generation. Each generator uses different spatial variation techniques and click-based interactivity, reflecting different design inspirations.
